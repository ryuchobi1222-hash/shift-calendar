# 希望休Webアプリ v1

スマホで使える、希望休入力用のシンプルなWebアプリです。

## 機能

- 月ごとのカレンダー表示
- 日付をタップして「出勤 / 希望休」を切り替え
- 土日は初期状態で希望休
- 希望休の日数を表示
- 希望休一覧をコピー
- 入力内容はスマホ内に保存

## GitHub Pagesで公開する方法

1. GitHubのリポジトリを開く
2. `index.html`、`style.css`、`app.js` をアップロード
3. Settings → Pages
4. Branchを `main` にする
5. `/root` を選んで Save
6. 少し待つとURLが発行されます

## iPhoneのホーム画面に追加する方法

1. Safariで公開URLを開く
2. 共有ボタンを押す
3. 「ホーム画面に追加」
4. 追加

これでアプリっぽく使えます。
